# SNHU Financial & Automation Portfolio

This repository contains data automation, database querying, and financial scripting solutions developed alongside my coursework at Southern New Hampshire University (SNHU). 

The goal of this project is to demonstrate how core business logic, economic principles, and accounting controls can be translated into functional, automated codebase solutions.

## Featured Scripts

### 1. Relational Ledger Queries (`database_practice.py`)
* **What it does:** Uses SQL logic within an embedded SQLite database engine to filter, query, and organize enterprise transaction tables.
* **Business Logic:** Automatically generates an aging AR report by filtering for "Unpaid" client accounts with values exceeding $2,500, sorted dynamically from highest risk to lowest.

### 2. Price Elasticity of Demand Calculator (`eco_calc.py`)
* **What it does:** Calculates consumer price sensitivity by evaluating absolute percentage shifts in quantities demanded relative to price fluctuations for ECO-201.
* **Business Logic:** Utilizes conditional loops to automatically categorize goods as *Elastic*, *Inelastic*, or *Unit Elastic* based on real-world market criteria.

## Core Technical Skills Demonstrated
* **Languages:** Python 3.12, SQL (Structured Query Language)
* **Frameworks/Tools:** SQLite3, Functional Programming, Relational Data Structures
* **Environment:** Visual Studio Code on Windows 11
