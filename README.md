# SNHU Microeconomics Automation Portfolio

This repository contains Python automation scripts developed alongside my coursework at Southern New Hampshire University (SNHU) for ECO-201 (Microeconomics). 

The goal of this project is to demonstrate how core economic and financial formulas can be translated into functional, automated codebase solutions.

## Featured Scripts

### 1. Price Elasticity of Demand Calculator (`eco_calc.py`)
* **What it does:** Calculates consumer price sensitivity by evaluating absolute percentage shifts in quantities demanded relative to price fluctuations.
* **Business Logic:** Utilizes conditional loops to automatically categorize goods as *Elastic*, *Inelastic*, or *Unit Elastic* based on real-world market criteria.

## Core Technical Skills Demonstrated
* **Language:** Python 3.12
* **Concepts:** Functional programming, conditional business logic processing, mathematical automation.
* **Environment:** Visual Studio Code on Windows 11
